# Security policy

Please report vulnerabilities privately through
[GitHub Security Advisories](https://github.com/jobiwanken0bi/atmosphere-account/security/advisories/new)
or email `contact@atmosphere.money`. Do not open a public issue for a suspected
vulnerability.

Include the affected route or component, reproduction steps, impact, and any
known mitigation. Avoid accessing data that is not your own and stop testing if
you encounter private user information.

The current `main` branch and production deployment are supported. Security
fixes are validated with the repository's checks, browser login E2E, and
production smoke suite before disclosure.
