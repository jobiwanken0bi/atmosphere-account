# Support

For reproducible bugs, feature proposals, and translation work, use the issue
forms on [GitHub](https://github.com/jobiwanken0bi/atmosphere-account/issues) or
open an issue on
[Tangled](https://tangled.org/@joebasser.com/atmosphere-account/issues).

Before opening an issue:

- Check the [documentation](./docs/ARCHITECTURE.md) and existing issues.
- Include the affected URL, environment, expected behavior, and actual result.
- Remove access tokens, cookies, private keys, database URLs, DIDs you do not
  own, and other personal data from logs and screenshots.

The public issue trackers are community support channels, not guaranteed
service-level support. Production status can be checked at
[`/api/health/ready`](https://atmosphereaccount.com/api/health/ready).

Suspected vulnerabilities and sensitive abuse reports must not be filed
publicly. Use [SECURITY.md](./SECURITY.md) or the private conduct-reporting path
in [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md).
